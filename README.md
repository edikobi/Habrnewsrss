# Агрегатор образовательного контента

Система для агрегации образовательного контента из различных источников (YouTube, Habr, Coursera) с персонализированными рекомендациями и отслеживанием прогресса обучения.

## Возможности системы

*   **Агрегация данных:** Сбор информации о курсах и статьях из Habr.
*   **Персонализация:** Система рекомендаций на основе интересов пользователя.
*   **Управление прогрессом:** Возможность отмечать изученные материалы и планировать обучение.
*   **Интерактивный CLI:** Удобный интерфейс командной строки для поиска и фильтрации контента.
*   **Экспорт данных:** Сохранение подборок в JSON/CSV форматы.

## Структура проекта

*   `src/` — Исходный код приложения.
    *   `api/` — Клиенты для взаимодействия с внешними API.
    *   `cli/` — Логика интерфейса командной строки.
    *   `core/` — Ядро системы: бизнес-логика и модели данных.
    *   `db/` — Слой работы с базой данных (SQLite).
    *   `services/` — Сервисы рекомендаций и обработки данных.
*   `data/` — Локальное хранилище данных и кэш.
*   `tests/` — Модульные и интеграционные тесты.
*   `config/` — Конфигурационные файлы.
